# アプリケーションのビルドと実行

実際にソースからビルドから試したい場合は下記の手順を実行ください。

ソースをGitHubからclone後、ルートフォルダで下記のコマンドを実行して下さい。※ Node.jsのインストールが必要です。

- 必要なモジュールのインストール
```
npm install
```

- ビルド
```
npm run build
```

- Electronアプリの実行
```
npm run start
```
